## Auto-release

# To trigger the auto-release workflow on Github (and Curseforge) you must change the version in the TOC file.

# A version can be :
- 1.2.3
- 1.2.3-stable
- 1.2.3-alpha
- 1.2.3-beta

# The version line must be as follow (taking spaces into account)
- "## Version: 1.2.3-type"
